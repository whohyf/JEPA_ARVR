# Human-Readable Chinese Notes

This directory contains Chinese companion notes written for human reading.

**Agent guidance:**

- Do not treat files in this directory as canonical experiment state.
- Prefer English docs under `docs/`, source code, and (for experiment history/metrics) the VJEPA2-EXP `logs/DASHBOARD.md` and branch cards.
- Read these Chinese notes only when the user explicitly asks for the Chinese explanation or asks to edit these files.
- If a Chinese note appears to contain a fact that is missing from the English docs, update the English canonical source first.
